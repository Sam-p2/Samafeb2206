# 📺 Samsung 2008 Startup Sound Simulator

A retro-style web project that recreates a **Samsung 2008 TV boot experience** with the original startup sound (via local MP3 file). Built using pure HTML, CSS, and JavaScript.

---

## 🎯 Features

- 🎬 Authentic Samsung 2008-inspired UI design
- 🔊 Plays real startup sound (MP3 local file)
- 🎚️ Volume control slider
- ⏻ Power On / Replay sound buttons
- 📺 TV-style animation and interface layout
- ⚡ Boot sequence status messages
- 💻 Fully responsive design (works on mobile & desktop)
